# CodeAlpha Basic Network Sniffer

## Overview

This project is a Python-based Network Sniffer developed as part of the CodeAlpha Cyber Security Internship. It captures and analyzes live network traffic using the Scapy library.

## Features

* Captures live network packets
* Displays Source IP Address
* Displays Destination IP Address
* Identifies TCP and UDP protocols
* Monitors network traffic in real time

## Technologies Used

* Python
* Scapy
* Npcap
* Windows

## How to Run

1. Install Scapy:
   pip install scapy

2. Run the program:
   python network_sniffer.py

## Sample Output

Source IP      : 192.168.31.147
Destination IP : 20.184.175.21
Protocol       : TCP
